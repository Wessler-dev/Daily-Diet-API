# 🥗 Daily Diet API

Uma API REST desenvolvida com **Python** e **Flask** para gerenciamento de refeições diárias. A aplicação permite que usuários registrem, consultem, atualizem e removam refeições, facilitando o acompanhamento da alimentação e o controle da dieta.

## 📖 Sobre o projeto

Neste projeto foi desenvolvida uma API completa para controle de dieta diária utilizando **Flask**. A aplicação possibilita o cadastro de refeições com informações detalhadas, como:

- 🍽️ Nome da refeição
- 📝 Descrição
- 📅 Data e horário
- ✅ Indicação se a refeição está dentro da dieta

Além do cadastro, a API oferece todas as operações de um **CRUD (Create, Read, Update e Delete)** para gerenciamento das refeições, utilizando um banco de dados para persistência das informações.

Este projeto foi desenvolvido com o objetivo de reforçar, na prática, conceitos importantes do desenvolvimento de APIs RESTful com Flask, como organização de rotas, manipulação de requisições HTTP, persistência de dados e estruturação de aplicações seguindo boas práticas.

---

## 🚀 Funcionalidades

- ✅ Cadastrar uma nova refeição
- 📋 Listar todas as refeições
- 🔍 Buscar uma refeição por ID
- ✏️ Atualizar uma refeição existente
- 🗑️ Excluir uma refeição
- 💾 Persistência dos dados em banco de dados
